This is a quick tool to estimate how much flame score one (magic) attack and 1% all stats is equivalent to for most classes in MapleStory.
One flame score is defined to be 1 point in your class main stat.
Note: This tool does not work for Xenons or Demon Avengers.
Once again, this tool provides a rough estimate. Please use a stat calculator for more accurate numbers.
